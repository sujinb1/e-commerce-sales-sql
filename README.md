# e-commerce-sales-sql

 E-Commerce Sales Analysis using SQL
*Project Overview

This project focuses on analyzing an e-commerce dataset using SQL to extract meaningful business insights. The goal is to understand customer behavior, product performance, and overall sales trends using real-world analytical queries.

The project is built using a relational database with tables such as customers, orders, order_items, and products.


*Database Structure

The dataset contains the following tables:

customers → Customer details (name, city, signup date)
products → Product details (name, category, price)
orders → Order information (customer, date)
order_items → Products purchased in each order


 *Key Business Questions Solved
1. Top Revenue Generating Products
2. Customer Lifetime Value (CLV)
3. Monthly Revenue Trend

Key Insights
Certain products contribute significantly more revenue than others.
A small group of customers contribute a large portion of total sales.
Monthly revenue trends help identify growth patterns and seasonal demand.
